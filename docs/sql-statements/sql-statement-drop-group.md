# DROP GROUP

Removes a database role.

## Synopsis

``` {#sql_command_synopsis}
DROP GROUP [IF EXISTS] <name> [, ...]
```

## Description

`DROP GROUP` is an alias for [DROP ROLE](DROP_ROLE.html).

## Compatibility

There is no `DROP GROUP` statement in the SQL standard.
